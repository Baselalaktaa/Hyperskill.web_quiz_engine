Implement a program that stores userRepo names. Your program should contain two endpoints:

  • POST /users: takes the query parameter name that represents the userRepo name and stores the userRepo name;
  • GET /users: returns an array of userRepo names in the order they were added. If there are no users returns an empty array.

In both cases, the server should respond with 200 (OK) status code.

Example 1: Response for a GET /users request:

[]

Example 2:

POST /users/?name=Alice
POST /users/?name=Kate
POST /users/?name=Someone

Example 3: Response for a GET /users request:

[
   "Alice",
   "Kate",
   "Someone"
]
